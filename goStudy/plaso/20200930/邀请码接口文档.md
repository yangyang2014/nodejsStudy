# 邀请码接口文档

## 1. 生成邀请码接口

- **url** /genInviteCode

- **请求类型** post

- **请求参数** 
无

- **响应参数** 

|字段名称  |字段类型 |是否必填|解释         |
|:-------:|:-------:|:-----:|:-----------:|
| code    |int   | 是    |0 表示正常，其他表示异常|
| obj |string    | 是    |六位邀请码|

- **示例**
http://localhost:8080/genInviteCode

`{"code":0,"obj":"308746"}`

# 2. 删除邀请码接口

- **url** /drop

- **请求类型** post

- **请求参数** 
  
|字段名称  |字段类型 |是否必填|解释         |
|:-------:|:-------:|:-----:|:-----------:|
| inviteCode    |string   | 是    |待删除的邀请码|

- **响应参数**

|字段名称  |字段类型 |是否必填|解释         |
|:-------:|:-------:|:-----:|:-----------:|
| code    |int   | 是    |0 表示正常，其他表示异常|
| obj    |string   | 否    |服务器提示信息|


- **示例**
http://localhost:8080/drop?inviteCode=186311

`{"code":0,"obj":""}`
